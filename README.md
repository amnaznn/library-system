# 📚 Library Management System

## 📌 Project Description
This is a simple Library Management System web application.

## 🚀 Features

- 🔐 Login system (hardcoded authentication)
- 📊 Dashboard with summary cards
- 📈 Data visualization using Chart.js
- 📚 Books management page (data table)
- 👥 Members management page (data table)
- 📱 Responsive design using Bootstrap 5
- 🧭 Navigation bar across all pages
